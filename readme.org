** Installation:

Clone this repository and make the location known to Emacs as follows:

#+BEGIN_SRC elisp
  (add-to-list 'custom-theme-load-path
               "~/path/to/naga-blue")
  (load-theme 'naga-blue t)

#+END_SRC
